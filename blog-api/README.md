# Blog API

## Proje Amacı
Backend geliştirme yeteneklerini göstermek için geliştirilmiş bir REST API projesidir.

## Özellikler
- Kullanıcı kayıt ve giriş
- JWT authentication
- Blog CRUD işlemleri
- Yorum sistemi

## Teknolojiler
- Java Spring Boot
- MySQL
- Spring Security
- Swagger

## Çalıştırma
1. MySQL'de blogdb oluştur
2. application.properties düzenle
3. Projeyi çalıştır
